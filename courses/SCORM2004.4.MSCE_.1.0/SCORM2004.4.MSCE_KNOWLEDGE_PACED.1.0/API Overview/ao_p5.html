API Overview (concluded)

The methods exposed by the API Implementation are divided into three categories.

Method Description
Session Methods Session methods are used to mark the beginning and end of a communication session between a SCO and a LMS through the API Instance.
Data-Transfer Methods Data-Transfer methods are used to exchange data model values between a SCO and a LMS through the API Instance.
Support Methods Support methods are used for auxiliary communications (e.g., error handling) between a SCO and a LMS through the API Instance.
