Data-Transfer Methods

A Sharable Content Object (SCO) uses data-transfer methods to direct the storage and retrieval of data to be used within the current communication session. The SCO uses these methods to transfer run-time data to and from the learning management system (LMS). For example, the LMS can use this data to help determine completion/mastery of activities and make sequencing and navigation decisions.

Upon completion of the content, learners will be able to identify the following for each data-transfer method:

  • Method Syntax
  • Description
  • Parameter
  • Return Value
